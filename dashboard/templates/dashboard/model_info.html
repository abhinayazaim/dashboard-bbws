⚙️ Arsitektur AI

Sistem ini mengimplementasikan arsitektur hybrid LSTM-Transformer (Long Short-Term Memory). LSTM dikonfigurasi untuk menangkap dependensi temporal sekuensial jangka panjang dari fluktuasi TMA historis, sementara lapisan Transformer menggunakan mekanisme self-attention untuk secara dinamis menimbang pengaruh anomali cuaca eksternal (curah hujan tinggi tiba-tiba) secara real-time.


Konfigurasi ini memastikan prediksi yang sangat responsif terhadap lonjakan debit air ekstrem yang tidak terprediksi secara linier. Model dilatih menggunakan {{ model_info.dataset_rows|default:"8558" }} baris data historis dengan window size {{ model_info.look_back|default:"90" }} time steps dan batch size {{ model_info.batch_size|default:"64" }}.
