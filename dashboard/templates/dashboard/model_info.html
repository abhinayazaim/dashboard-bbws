memory Arsitektur AI

Sistem ini mengimplementasikan arsitektur hybrid LSTM-Attention (Long Short-Term Memory). LSTM dikonfigurasi untuk menangkap dependensi temporal sekuensial jangka panjang dari fluktuasi TMA historis, sementara lapisan Attention Layer menggunakan mekanisme feature-wise attention untuk secara dinamis menimbang pengaruh faktor eksternal (curah hujan tinggi, debit sungai) secara real-time. Konfigurasi ini memastikan prediksi yang sangat responsif terhadap lonjakan TMA ekstrem yang tidak terprediksi secara linier.
