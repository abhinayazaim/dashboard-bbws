Sistem Prediksi Hidrologi Bendungan Bajulmati dirancang untuk memberikan estimasi Tinggi Muka Air (TMA) menggunakan model AI hybrid LSTM-Attention.

Cara Penggunaan:

  • Gunakan menu Input Manual untuk memprediksi data tunggal secara real-time.
  • Gunakan menu Unggah Batch untuk memprediksi banyak data sekaligus menggunakan file CSV.
  • Amati Kinerja Model di Dashboard utama untuk melihat komparasi data prediksi dan aktual.
